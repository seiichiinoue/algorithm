### ダイクストラ法
#### ダイクストラ法とは
[ダイクストラ法](https://ja.wikipedia.org/wiki/%E3%83%80%E3%82%A4%E3%82%AF%E3%82%B9%E3%83%88%E3%83%A9%E6%B3%95) (Dijkstra's Algorithm) は最短経路問題を効率的に解くグラフ理論におけるアルゴリズム．スタートノードからゴールノードまでの最短距離とその経路を求めることができる．
